# 충북대를 내 손 안에
충북대학교는 92.7만m<sup>2</sup>로 국내에서 20위권의 면적 크기를 갖고 있다!<br>
충북대학교의 건물 수는 130개가 넘으며, 학교 면적 대비 많은 건물이 있다!

그 건물들의 정보와 빠른 길 안내를 도와줄 &lceil;선-배&rfloor;가 **\<Campus Map\>** 안에 들어있다!!!

## 데모 영상
추가 예정
 
## 사용된 기술 
- Android Studio
- Room DB
- Dijkstra Algorithm
